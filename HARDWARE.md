# StarVR One Specification

| Category             | Specification                                                                |
|----------------------|------------------------------------------------------------------------------|
| Display              | 2 × 4.77" AMOLED, 8-bit, 100% NTSC                                           |
| Resolution           | 1792 × 2240 per eye (stretched to 1920x2240 via software)                    |
| Refresh Rate         | 72 Hz, 75 Hz or 90 Hz                                                        |
| Field of View (FOV)  | 210° Horizontal / 130° Vertical                                              |
| Eye Tracking         | Integrated Tobii EyeChip                                                     |
| IPD Adjustment       | Automatic via Eye-Tracking                                                   |
| Tracking             | SteamVR Tracking 2.0 on non-XT and PhaseSpace or OptiTrack on XT variant     |
| Connectivity         | 2 × USB Type-C (0.9 m), 2 × Type-C extensions, 3.5 mm audio jack             |
| Port Requirements    | 2 × DisplayPort, 2 × USB 2.0                                                 |
| Weight               | ~450 g                                                                       |
| Audio                | 3.5 mm headphone jack with microphone                                        |
